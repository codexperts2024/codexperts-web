# Week 6 — QA & Launch

## Goals

Full end-to-end QA across all user roles, fix remaining bugs, and go live at `codexperts.ca`.
This sprint is about stability and delivery, not new features.

---

## Deliverables

### Quality Assurance
- **Full flow QA** — test every user role scenario end-to-end (see checklist below)
- **Bug fixes** — each team member resolves issues found in their area
