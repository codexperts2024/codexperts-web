# Week 6 — QA & Launch

## Goals

Full end-to-end QA across all user roles, fix remaining bugs, and go live at `codexperts.ca`.
This sprint is about stability and delivery, not new features.

---

## Deliverables

### Quality Assurance
- **Full flow QA** — test every user role scenario end-to-end:
  - Public visitor browsing all public pages
  - New user signing in → pending screen → Admin approval → member access
  - Member submitting code → viewing submissions
  - Admin scanning a QR session → member checking in
  - Admin managing users, roles, and problems
- **Bug fixes** — each team member resolves issues found in their area

### Production Deployment
- **Vercel `main` branch** — connected to production environment; custom domain configured
- **Railway FastAPI** — production environment variables set, CORS updated for `codexperts.ca`
- **Domain connection** — `codexperts.ca` pointed to Vercel production deployment

### Documentation
- **README final update** — Agile Process sprint log completed through Week 6
- **docs/ cleanup** — all specs and guidelines up to date

- [ ] All 5 user flows tested and passing on production
